ARIA Toggle Blocks

Example of a progressively enhanced content toggle area. The toggled content is controlled by a link that is progressively enhanced into a button, so the toggle action can function via CSS, in instances where JavaScript may not be enabled.

Please review the following examples to see how this component may be utilized, and the options it contains.
